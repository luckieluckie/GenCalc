# GenCalc

GenCalc is an AI-powered calculator that lets users draw mathematical expressions on a canvas and get step-by-step solutions using Google's Gemini API.

## Features
- Freehand canvas drawing
- Multiple color tools
- Undo & reset
- AI-generated explanation + final answer
- Clean, distraction-free UI

## Tech Stack
- React
- HTML Canvas
- Node.js
- Express
- Google Gemini API

## Project Structure
GenCalc/
- ├── frontend/
- └── backend/
# Backend
- cd backend
- npm install
- node server.js

# Frontend
- cd frontend
- npm install
- npm start

# Important Point
- This project uses Gemini free-tier API. For demo purposes, usage is limited and the key can be rotated when quota is exhausted.
